HDR

  • 實作論文 Recovering High Dynamic Range Radiance Maps from Photographs. In SIGGRAPH 97, August 1997.

概念

  • 此為相機獲得 ZZZ 的流程

    而我們的目標就是從 ZZZ 還原到 EEE，再將多張不同曝光時間的圖結合，獲得 HDR 的圖片
  • 每一家相機所使用的 Response Function 都不一樣，而我們的目標就是要還原這個 Response Function 的反函數，即可將 ZZZ 還原到 EEE
